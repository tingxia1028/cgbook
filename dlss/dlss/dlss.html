input

  • depth buffer
  • exposure value (for HDR)
  • previous output buffer(optional) only RGBA16F

pipeline

mip-map bias

我们引擎里的lod bias， 原来lod 1.2， bias 0.5， 则最终采用mip map level 为1.7. 又因为DLSS是渲了一个更小的分辨率，所以尽量让LOD减小以增加精度。

motion vector

  • pixel space not uv space，可以使用给一个motion vector scale 为render resolution来缩放
  • need to be accurate every object must have one，(1, 1)怕是不兼容
